Object Recognition Robot
Skills: Object Recognition, Python, Laser Cutter, proximity Sensor, 3D Print, Raspberry Pi
Terminal Code

In this project, a Trainable Machine algorithm was trained to recognize 6 different objects and proximity sensor code was written to decide which way to turn once a specified distance was measured away from the object. Then the object is recognized and the turn direction decision is made. This was done to have the iRobot Create 3 navigate an obstacle course with the objects used in the recognition algorithm used as checkpoints.
