Autonomous Pancake Maker
Skills: PID Control, DC Motor, Rapid Prototyping, Raspberry Pi Camera Module 2, 3D Printing, H-Bridge, Raspberry Pi, HTML, Python, Airtable
Terminal Code

In my Robotics and Mechatronics course the class was challenged to create an autonomous pancake making robot. We created a robot which can functionally follow the line using IR sensors and stop at stations by using a color sensor. The device communicated with different station mechanisms of the pancake making process through Airtable to control timing and quality of each operation. The output was 2 edible pancakes with selected toppings specific to the users input order on the front end website.
